I build stuff with AI and code. Sometimes it even works.

Two things I care about:
- Making AI agents safer so they don't do dumb (or dangerous) things
- Using tech to help people see where public money goes

### What I'm working on

- **[aiagentobservatory.org](https://aiagentobservatory.org)** — Research on Agentic AI risks and security
- **[mcpwn](https://github.com/Carlos-Projects/mcpwn)** — Active security testing framework for MCP servers (command injection, SSRF, tool poisoning, RCE blind detection). Like a pentesting toolkit but for AI agents.
- **[agentgate](https://github.com/Carlos-Projects/agentgate)** — Policy-based firewall for AI agents browsing the web. Blocks malicious sites, enforces navigation policies, honeypot for attackers.
- **[palisade-scanner](https://github.com/Carlos-Projects/palisade-scanner)** — Scans websites for prompt injection, hidden instructions, and adversarial content targeting AI agents.
- **[mcpscope](https://github.com/Carlos-Projects/mcpscope)** — Unified dashboard for MCP/A2A security scanning results. Aggregates findings from multiple scanners.
- **[veeduria](https://github.com/Carlos-Projects/veeduria)** — Algorithmic auditing platform for public procurement. Uses AI to monitor government contracts and detect irregularities.
